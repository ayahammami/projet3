About this page :

this is a great place to talk about your webpage. This template is purposeflly unstyled so you can use it as a boilerplate or starting point for your own landing page designs! This template features.

  • clickable nav links that smooth scroll to page sections
  • Responsive behavoir when clicking nav links perfect for a one page website
  • Bootstrap's scrolppy feature which highlights which section of the page you're on in the navbar
  • Minimal custom so you are free to explore your own unique options
